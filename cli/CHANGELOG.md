## [0.1.0] - 2024

### Added
- Initial CLI implementation with commander.js
- Wallet management commands:
  - `iln wallet generate` for creating new keypairs
  - `iln wallet import` for importing existing secret keys
  - `iln wallet show` for displaying balances and public key
  - `iln wallet fund` for testnet XLM from Friendbot
  - `iln wallet list` for listing saved profiles
- Invoice commands:
  - `iln submit` with flag mode, interactive prompts, and `--dry-run`
  - `iln cancel` with state guard and confirmation prompt
  - `iln fund` with confirmation prompt and `--yes` skip flag
  - `iln status` with rich output, `--json`, and `--watch` flags
  - `iln submit-receipt` for receipt submission
- Marketplace commands:
  - `iln marketplace` with sort and filter flags
- Reputation commands:
  - `iln reputation` for checking on-chain reputation scores
  - `iln reputation --address G...` for specific addresses
  - `--json` flag for machine-readable output
- Configuration commands:
  - `iln config set network <testnet|mainnet>`
  - `iln config set rpcUrl <url>`
- Export command:
  - `iln export <file.json>` for exporting invoices
- Shell completion support:
  - `iln completion bash`
  - `iln completion zsh`
  - `iln completion fish`
- Helper utilities:
  - State badge formatter
  - Time until expiry formatter
  - Invoice detail formatter
  - Mini status timeline builder
  - Receipt table builder
  - BPS to yield percentage converter
  - State validation for cancellable invoices
  - Confirmation message formatter

### Tests
- Comprehensive unit tests for all helper functions
- E2E tests for all commands
- Interactive prompt mode tests with mocked prompter
- `--dry-run` mode tests
- `--watch` mode tests for status command
- `--json` flag tests for JSON output validation
- State guard and error path tests

- Scripting and automation examples
