## [0.1.0] - 2024

### Added
- Initial SDK implementation with ILNClient builder pattern
- Freighter wallet adapter for browser integration
- Core invoice methods:
  - `submitInvoice` with full type safety and error mapping
  - `fundInvoice` with multi-token support
  - `markPaid` with partial payment support
  - `cancelInvoice` for invoice cancellation
  - `disputeInvoice` for dispute handling
  - `transferLPPosition` for LP position transfers
- Query methods:
  - `getInvoice` for fetching invoice details
  - `listInvoices` for batch invoice retrieval
  - `getReputation` for on-chain reputation scores
  - `getContractStats` for contract statistics
- Governance SDK methods:
  - Proposal creation and management
  - Voting and delegation
  - Proposal execution
- TokenRegistry for multi-token support (USDC, XLM, EURC)
- Centralized input validation utilities
- Comprehensive JSDoc documentation for all exported functions and types
- Automated npm release workflow with dual ESM/CJS output
- Integration test suite running against Stellar testnet
- TypeScript type definitions for all contract interactions
- Error mapping from Soroban error codes to descriptive messages

### Documentation
- Comprehensive SDK README with usage guide and examples
- Integration guide with tested code examples
- TypeDoc API documentation generation
